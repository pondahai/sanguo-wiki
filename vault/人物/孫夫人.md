# 孫夫人

### 生平概述

[[孫夫人]]為[[孫權]]之妹，自幼好觀武事，性格剛烈。[[孫權]]與[[周瑜]]為實施美人計，將其許配給[[劉備]]，二人於洞房中結為夫妻，兩情歡洽。[[孫夫人]]雖有武備，但後聽從[[劉備]]之言，令侍婢解劍伏侍。

[[劉備]]以祭祖為名不告而別，[[孫夫人]]同意並同其出城，與[[趙雲]]會合後離開南徐。途中遭遇[[徐盛]]、[[丁奉]]、[[陳武]]、[[潘璋]]攔截，[[孫夫人]]憑藉[[吳國太]]慈旨及威勢，叱退眾人，護送[[劉備]]先行，自己與[[趙雲]]斷後，成功脫離[[周瑜]]的追擊。

[[孫夫人]]後因[[吳國太]]病危密書，將[[阿斗]]藏於車中不辭而別回東吳。途中與[[趙雲]]發生衝突，喝令侍婢揪捽[[趙雲]]並奪回[[阿斗]]，威脅若不放行便投江而死。回吳後，[[孫權]]因[[周善]]被殺大怒欲攻荊州。[[孫夫人]]在吳期間，[[諸葛瑾]]曾以送回[[孫夫人]]為條件要求講和。

*   自幼好觀武事，令侍婢擊劍，後聽[[劉備]]之言命侍婢解劍伏侍（第55回）。
*   笑稱廝殺半生不懼兵器（第55回）。
*   與[[劉備]]成親，兩情歡洽（第55回）。
*   勸慰煩惱的[[劉備]]，表示任君所之，妾當相隨（第55回）。
*   同意[[劉備]]以祭祖為名不告而去（第55回）。
*   同[[劉備]]出城，與[[趙雲]]會合後離了南徐（第55回）。
*   被[[徐盛]]、[[丁奉]]攔截時，叱從人推車直出，捲起車簾親喝[[徐盛]]、[[丁奉]]（第55回）。
*   怒罵[[周瑜]]逆賊，喝令推車前進，[[徐盛]]、[[丁奉]]放其通過（第55回）。
*   再次叱退[[陳武]]、[[潘璋]]，表示奉[[吳國太]]慈旨回荊州，罵得四人面面相覷而退（第55回）。
*   令推車便行，讓[[劉備]]先行，自己與[[趙雲]]當後（第55回）。
*   [[孫夫人]]車馬僕從停住於山頂之上，令[[周瑜]]見狀氣憤（第56回）。
*   [[孫權]]將妹妹嫁給[[劉備]]，與[[劉備]]結為親家（第56回）。
*   [[孫夫人]]見密書說[[吳國太]]病危，便將[[阿斗]]藏在車中，不辭而別回東吳（第61回）。
*   [[孫夫人]]在船上喝問[[趙雲]]何故無禮，並怒斥[[趙雲]]是帳下武夫（第61回）。
*   [[孫夫人]]喝侍婢向前揪捽[[趙雲]]，並奪[[阿斗]]（第61回）。
*   [[孫夫人]]威脅若不放她回去，情願投江而死（第61回）。
*   [[孫夫人]]回吳後，具說[[張飛]]、[[趙雲]]殺了[[周善]]，截江奪了[[阿斗]]（第61回）。
*   [[孫權]]因[[孫夫人]]回吳且[[周善]]被殺，大怒欲報仇並起軍攻取荊州（第61回）。
*   [[孫夫人]]已南歸，[[法正]]認為她未必再來（第77回）。
*   [[孫夫人]]一向思歸，[[諸葛瑾]]以此作為講和條件之一要求送回（第82回）。
*   [[孫夫人]]在吳，聞猇亭兵敗，訛傳[[劉備]]死於軍中（第84回）。
*   [[孫夫人]]驅車至江邊，望西遙哭，投江而死（第84回）。
