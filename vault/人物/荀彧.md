# 荀彧

### 生平概述

荀彧，字文若，早年曾效力於[[袁紹]]，後棄紹投奔[[曹操]]，被曹操譽為「吾之子房」，並任命為行軍司馬。在[[曹操]]早期發展階段，荀彧留守後方，鎮守鄄城、范縣、東阿三縣，並與[[程昱]]設計死守此三處，為[[曹操]]保全根基。隨後，荀彧建議[[曹操]]奉天子以從衆望，定都許都，並因此被任命為侍中尚書令，成為[[曹操]]集團中樞的核心謀士。

在[[曹操]]征戰四方期間，荀彧多負責留守許都，調遣兵將並籌措糧草。他多次為[[曹操]]提供戰略建議，包括擊破黃巾餘黨、圖謀呂布、對抗[[袁紹]]等，並曾提出「二虎競食之計」與「驅虎吞狼之計」等權謀策略。荀彧在[[曹操]]集團中地位崇高，不僅參與軍事決策，亦在政治禮制上發揮影響力，如諫阻[[曹操]]殺害[[禰衡]]與[[脂習]]，以及反對[[曹操]]接受魏公之位及九錫。

晚年，荀彧因反對[[曹操]]進爵魏公，已知[[曹操]]有殺己之心，遂託病止於壽春。[[曹操]]送來空盒，荀彧會意其意，遂服毒而亡。[[曹操]]得知後，命厚葬荀彧，諡曰敬侯。

### 重要事蹟

*   棄[[袁紹]]投奔[[曹操]]，被稱為「吾之子房」，任行軍司馬，並推薦[[程昱]]。（第10回）
*   領軍三萬，與[[程昱]]設計死守鄄城、東阿、范縣三處。（第10回、第11回）
*   諫阻[[曹操]]攻打徐州，指出棄兗州取徐州是棄大就小、去本求末。（第12回）
*   建議[[曹操]]東略陳地，擊破汝南、潁川黃巾餘黨以就食。（第12回）
*   建議[[曹操]]奉天子以從衆望，認為許都屬土正合董昭、王立之言。（第14回）
*   被[[曹操]]封為侍中尚書令。（第14回）
*   向[[曹操]]提出「二虎競食之計」。（第14回）
*   在「二虎競食之計」失敗後，提出「驅虎吞狼之計」。（第14回）
*   勸[[曹操]]殺[[劉備]]，認為不早圖後必為患。（第16回）
*   建議[[曹操]]加官賜賞[[呂布]]，令其與[[劉備]]解和以阻[[張繡]]。（第16回）
*   勸[[曹操]]暫回許都待來春麥熟再圖[[袁術]]。（第17回）
*   被[[曹操]]留鎮許都調遣兵將。（第17回）
*   差人報知[[曹操]][[劉表]]助[[張繡]]屯兵安衆截斷歸路。（第18回）
*   星夜馳書報知[[曹操]][[袁紹]]欲興兵犯許都。（第18回）
*   拜服[[曹操]]在安衆必勝的計策。（第18回）
*   認同[[郭嘉]]的十勝十敗論。（第18回）
*   建議[[曹操]]先約[[劉備]]，待其回報後方可動兵攻打[[呂布]]。（第18回）
*   認同[[郭嘉]]決水淹下邳的計策。（第19回）
*   入見[[曹操]]，表示天子認[[劉備]]為叔恐無益於[[曹操]]。（第20回）
*   建議[[曹操]]赦免[[朱靈]]、[[路昭]]，認為權歸[[劉備]]二人亦無奈何。（第21回）
*   建議[[曹操]]寫書與[[車胄]]，讓[[車胄]]在內部圖謀[[劉備]]。（第21回）
*   反驳[[孔融]]，認為[[袁紹]]无用，不必議和。（第22回）
*   分析[[袁紹]]手下謀士性格缺陷，預測必生內變。（第22回）
*   評價[[顏良]]、[[文醜]]匹夫之勇，一战可擒。（第22回）
*   勸阻[[曹操]]不要因[[禰衡]]辱罵而殺他。（第23回）
*   分析風向和旗幟顏色，預判[[劉備]]今夜必來劫寨。（第24回）
*   建議[[曹操]]速取下邳，否則恐被[[袁紹]]所竊。（第24回）
*   反對[[曹操]]退軍，擔心有詐。（第25回）
*   建議[[曹操]]不教[[關羽]]立功，[[關羽]]未必便去。（第25回）
*   密告[[曹操]][[關羽]]常有歸[[劉備]]之心，建議不可頻令出征。（第26回）
*   留守許都。（第30回）
*   回信建議[[曹操]]畫地而守，扼其喉使[[袁紹]]軍情見勢竭，斷不可失此用奇之時。（第30回）
*   受[[曹操]]急令，在許昌措辦糧草，星夜解赴軍前接濟。（第30回）
*   寫信告知[[曹操]][[劉備]]襲擊許昌的消息。（第31回）
*   被[[曹操]]留下守衛許都。（第31回）
*   建議[[曹操]]暫緩南征，待半年後再一鼓而下[[劉表]]、[[孫權]]。（第34回）

*   諫阻[[曹操]]天寒未可用兵並建議姑待春煖長驅大進。（第37回）
*   諫阻[[夏侯惇]]輕敵，指出[[劉備]]更兼[[諸葛亮]]為軍師，不可輕敵。（第39回）
*   被[[曹操]]留下守許昌。（第40回）

*   被[[曹操]]提及，作為[[荀攸]]欲效仿的對象（暗示[[荀彧]]曾反對[[曹操]]進爵魏公）。（第66回）

### 人物關係

*   **主公**：[[曹操]]
*   **部下**：[[程昱]]（荀彧推薦）、[[朱靈]]、[[路昭]]（荀彧建議赦免）
*   **親屬**：[[荀惲]]（子）
*   **盟友**：[[劉備]]（荀彧曾建議聯劉或圖劉，關係複雜，但清單中提及聯劉解和、認劉備為叔等事）
*   **對手**：[[袁紹]]、[[呂布]]、[[袁術]]、[[張繡]]、[[劉表]]、[[孔融]]
