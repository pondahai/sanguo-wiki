# 黃祖

### 生平概述

[[黃祖]]為[[劉表]]麾下將領，初受命率領江夏軍隊作為前驅部隊。在[[孫堅]]攻打江夏時，黃祖憑藉江邊地利佈置弓弩手伏擊，一度迫使孫堅軍撤退，並曾揚鞭怒罵江東軍隊。然而隨後孫堅反擊，黃祖大敗，棄守樊城退往鄧城，最終在野戰中再次被擊敗，僅能收集殘兵返回向[[劉表]]匯報戰況。

[[孫堅]]陣亡後，[[黃祖]]與[[蒯越]]、[[蔡瑁]]等人分頭出兵攻擊江東軍隊，導致對方大亂。但在後續交戰中，[[黃祖]]遭[[黃蓋]]水軍迎頭痛擊，戰況不利下被生擒。經[[劉表]]交涉，雙方交換人質，[[黃祖]]得以被釋放歸還[[劉表]]處。

晚年時期，[[黃祖]]駐守江夏，性格貪婪且昏庸，對百姓官吏橫徵暴斂，導致民心怨恨，軍事裝備荒廢，軍紀鬆散。面對[[孫權]]的大規模征討，他任命[[蘇飛]]為大將，[[陳就]]、[[鄧龍]]為先鋒出戰，但結局慘敗。最終在逃亡途中，[[黃祖]]於東門外遭遇昔日部下[[甘寧]]攔截，被其背射中箭墜馬，隨即被斬首示眾。

### 重要事蹟

*   受[[劉表]]命令，領江夏之兵為前驅，並在江邊埋伏弓弩手伏擊[[孫堅]]軍隊。（第7回）
*   在[[孫堅]]進攻時，因弓箭放盡而被迫退走，隨後棄守樊城，敗退至鄧城。（第7回）
*   在野外佈陣迎戰[[孫堅]]，並指使[[張虎]]搦戰，戰敗後丟棄頭盔戰馬混入步兵逃命。（第7回）
*   收集敗兵返回見[[劉表]]，報告[[孫堅]]勢力難以抵擋的情況。（第7回）
*   在[[孫堅]]死後，與[[蒯越]]、[[蔡瑁]]分頭引兵攻擊江東諸軍，造成對方混亂。（第7回）
*   被[[黃蓋]]水軍迎擊並生擒，後經[[劉表]]建議與[[孫堅]]屍首交換而被釋放。（第7回、第8回）
*   受[[劉表]]命令，負責堅守隘口。（第18回）
*   與[[禰衡]]共飲期間，因遭受侮辱而將[[禰衡]]斬殺。（第23回）
*   面對[[孫權]]的征討，認為[[甘寧]]是劫江之賊而不重用，對待[[甘寧]]態度輕薄。（第38回）
*   因年老昏昧、貪圖財利、侵擾吏民以及軍備廢弛，導致人心離散。（第38回）
*   在對抗[[孫權]]的戰役中，任命[[蘇飛]]為大將，[[陳就]]、[[鄧龍]]為先鋒，集合江夏全軍出戰。（第38回）
*   戰役中大敗，主帥[[蘇飛]]被[[潘璋]]生擒，自身處於勝負未分的狀態。（第38回）
*   兵敗後放棄江夏，向荊州方向逃跑。（第39回）
*   在東門外被[[甘寧]]攔截，背部中箭落馬，隨後被斬首。（第39回）
